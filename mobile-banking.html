Mobile Banking Application

Mobile UI for personal finance management

Overview

This project is a simple mobile banking-style application built as a school project using React Native and Expo. It allows users to track daily income and expenses and instantly see a summary of their financial situation.

Conceptualisation

The idea was to design a clean, blue-themed interface similar to a real banking app while focusing on core functionality. The app uses three main screens: Home (transaction list), Add Transaction and Summary.

Planning

  • Defined three main screens: Home, Add Transaction, Summary
  • Planned a simple data model: title, amount, type
  • Designed a consistent blue, card-based UI
  • Used in-memory state for instant UI updates

Development & Implementation

Built using React Native with Expo and JavaScript. Navigation is handled using a stack navigator. Transaction data is stored in state and shared across screens through navigation parameters.

Key Features

  • Add, Edit and Delete transactions
  • Summary screen with income, expense and balance
  • Clear surplus or deficit feedback
  • Friendly empty-state guidance
  • Consistent blue banking theme

Challenges & Solutions

Calculating totals accurately was the main challenge. This was solved using useMemo to derive totals from the transaction list efficiently and reliably.

Reflection

This project strengthened my understanding of shared state management, mobile UI design and presenting calculated data clearly. Future improvements could include persistence and data visualisation.
